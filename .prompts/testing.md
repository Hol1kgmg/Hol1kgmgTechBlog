# テストプロンプト

## テスト実行

```prompt
以下のコマンドでテストを実行してください：

```bash
# テストの実行
npm run test
# または
yarn test

# テストの監視モード
npm run test:watch
# または
yarn test:watch

# カバレッジレポートの生成
npm run test:coverage
# または
yarn test:coverage
```
```
