# フロントエンド開発プロンプト

## コンポーネント作成

```prompt
以下の要件に基づいて、[コンポーネント名]コンポーネントを作成してください：

1. 機能要件
- [具体的な機能要件を記述]

2. 技術要件
- Next.js 14
- Panda CSS
- TypeScript
- [その他の必要な技術]

3. ディレクトリ構造
- [FSDに基づいた配置場所]

4. 必要なprops
- [propsの型定義]

5. スタイリング要件
- [デザイン要件]
- [レスポンシブ対応]
- [アクセシビリティ]

6. テスト要件
- [単体テスト]
- [統合テスト]
```
