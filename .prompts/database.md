# データベース設計プロンプト

## テーブル設計

```prompt
以下のテーブルを設計してください：

1. テーブル名
- [テーブルの目的]

2. カラム定義
- [カラム名]: [型], [制約]
- [インデックス]
- [外部キー]

3. マイグレーション
- [アップマイグレーション]
- [ダウンマイグレーション]

4. シードデータ
- [初期データ]

5. テストデータ
- [テストケース]
```

## マイグレーション

```prompt
以下のマイグレーションを実装してください：

1. マイグレーション内容
- [変更内容]
- [影響範囲]
- [ロールバック]

2. データ移行
- [データ変換]
- [整合性]
- [パフォーマンス]

3. インデックス
- [インデックス設計]
- [最適化]
- [メンテナンス]

4. 制約
- [外部キー]
- [ユニーク制約]
- [チェック制約]

5. テスト
- [マイグレーションテスト]
- [ロールバックテスト]
```
