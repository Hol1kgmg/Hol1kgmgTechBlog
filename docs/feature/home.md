# ホームページ

## 概要
ホームページは、ブログの顔となるランディングページです。未認証ユーザーと認証済みユーザーの両方に適した情報を提供し、ブログの魅力を伝えるとともに、主要な機能へのアクセスを提供します。

## ディレクトリ構造
本機能の実装は、[ディレクトリ構造](../directory-structure.md)の規約に従って行います。

## 機能一覧

### 1. ヒーローセクション
- ブログのメインキャッチコピー
- 簡潔な説明文
- 新規登録/ログインボタン（未認証ユーザー向け）
- ダッシュボードへのリンク（認証済みユーザー向け）

### 2. 新着記事セクション
- 最新の記事一覧（最大5件）
- 記事のサムネイル画像
- タイトル
- 公開日
- 著者情報
- カテゴリータグ

### 3. 人気記事セクション
- 閲覧数の多い記事一覧（最大5件）
- 記事のサムネイル画像
- タイトル
- 閲覧数
- いいね数
- カテゴリータグ

### 4. カテゴリー一覧
- 主要カテゴリーの表示
- カテゴリーごとの記事数
- カテゴリーページへのリンク

### 5. 著者紹介
- 主要な著者のプロフィール
- 著者の最新記事へのリンク
- フォローボタン（認証済みユーザー向け）

## 画面構成

```
+------------------------+
|        ヘッダー        |
+------------------------+
|    ヒーローセクション   |
+------------------------+
| 新着記事 | 人気記事    |

+---------+------------+

+------------------------+
```

### コンポーネント構成
1. ヘッダー
   - ロゴ

   - 検索バー

2. ヒーローセクション
   - メインキャッチコピー
   - 説明文
   - CTAボタン

3. 記事セクション
   - 記事カード
   - ページネーション
   - フィルター

4. カテゴリー一覧
   - カテゴリーカード
   - 記事数表示

5. 著者紹介
   - プロフィールカード
   - 最新記事リンク

## 技術的な実装

### 使用コンポーネント
- `pages/index.tsx`: メインページコンポーネント
- `widgets/HomeHeader`: ヘッダーウィジェット
- `widgets/HeroSection`: ヒーローセクションウィジェット
- `features/articleList`: 記事一覧機能
- `features/categoryList`: カテゴリー一覧機能
- `features/authorList`: 著者一覧機能

### データフロー
1. ページロード時に必要なデータを取得
   - 新着記事
   - 人気記事
   - カテゴリー情報
   - 著者情報

2. 認証状態に基づく表示制御
   - 未認証ユーザー: 新規登録/ログインボタンを表示
   - 認証済みユーザー: ダッシュボードリンクを表示

## アクセス制御
- パブリックページとして誰でもアクセス可能
- 認証状態に応じて表示内容を変更

## パフォーマンス最適化
